This is my project for Task Tracker

Components

My task management project allows users to efficiently organize their tasks. Users can effortlessly add new tasks with due dates, mark task statuses, and easily delete tasks. This streamlined system enhances task management, providing a user-friendly experience for organizing and tracking responsibilities.

Details

The entry form facilitates users in adding tasks with due dates. Functional buttons enable users to delete or modify task statuses anytime. The home page serves as a welcoming hub, offering easy access to the task page. This user-centric design ensures seamless task entry, management, and navigation.

Look and Feel

The home page welcomes users with well-organized text and buttons. The tasks page boasts an appealing entry form and aesthetically pleasing task boxes for efficient task management. A simple header features a logo and three links to different pages, while the footer includes minimal contact and copyright information. Smooth animations enhance the user experience during page transitions.
